Custom Windows Keyboard
=======================
Built with `Microsoft Keyboard Layout Creator <http://msdn.microsoft.com/en-us/goglobal/bb964665.aspx>`__.

Plain US (without dead keys) plus umlauts: ``äöüÄÖÜß€``.
Get them with ``AltGr + <ancestor>``, e.g. ``AltGr + s`` --> ``ß`` or ``AltGr + O`` --> ``Ö``.

``felixhum.msklc`` is the source file.

Download: :download:`felixhum.zip`.

Install: setup.exe

Remove: Add/Remove Programs
